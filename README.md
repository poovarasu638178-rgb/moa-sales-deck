# Mall of America — Interactive Sales Deck

Cinematic interactive sales deck for Mall of America commercial leasing, sponsorship and event booking.

## Tech Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS  
- Framer Motion
- Lucide React

## Setup
npm install
npm run dev

## AI Tools Used
- Google Antigravity 2.0 (primary build agent)
- Claude AI (architecture, prompts, strategy)
- Unsplash (visual assets)

## Design Decisions
- Luxury dark theme #0a0a0a with gold accents
- Non-linear floating dot navigation
- Ken Burns cinematic hero effect
- Framer Motion scroll-triggered animations
- Apple and Tesla inspired minimal luxury UI
